NC News API

This API mimics that found on a social news aggregator (think Reddit). It provides functionality to access and manipulate articles, comments, topics and users.

Check out a list of available endpoints, or visit the project on GitHub for more information.

Method Endpoint Description
GET /api Serves a JSON representation of all the available endpoints
GET /api/topics Serves an array of all topics
GET /api/articles Serves an array of all articles
GET /api/articles/:article_id Serves an individual article, specified by the article_id
PATCH /api/articles/:article_id Update the vote count for a specific article
GET /api/articles/:article_id/comments Serve an array of comments for the specified article
POST /api/articles/:article_id/comments Add a new comment for the specified article
DELETE /api/comments/:comment_id Remove the comment with the specified id
GET /api/users Serves an array of all users
GET /api/users/:username Serves an individual user, specified by the username users
PATCH /api/comments/:comment_id Update the vote count for a specific comment
POST /api/articles Add a new article
